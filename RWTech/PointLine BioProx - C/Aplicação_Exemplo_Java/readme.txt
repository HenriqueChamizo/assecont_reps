A aplicação "SimuladorIntegracaoREP.jar" foi desenvolvida para auxiliar os desenvolvedores no processo de integração com os relógios de ponto da linha iPointline RWTECH.

Antes de executá-la, é necessário instalar o Java e, posteriormente, substituir os arquivos "local_policy.jar" e "US_export_policy.jar" no diretório "lib\security" do Java instalado.
